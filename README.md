# weather-app
Weather App. Written in TypeScript using React.
[Demo](https://public-weather-app.netlify.app)

## How to use
Clone
```bash
git clone git@github.com:muhammednajm/weather-app.git
```

Install
```bash
cd weather-app && npm i
```

Build first (for production)
```bash
npm run build
```

Preview
```bash
npm run preview
```

Development mode
```bash
npm run dev
```
